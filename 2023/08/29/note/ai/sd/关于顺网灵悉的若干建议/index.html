# 关于顺网灵悉的若干建议

# 概述

目前，顺网灵悉的测试版本已经上线，开放的功能包括 AI 聊天、写作、图像生成等。我在此针对 UI、文生图、文生文等功能提出一些建议

# UI 方面

# icon 和 UI 主题色不匹配

icon 和 UI 颜色不匹配会有一定割裂感，市面上较为有代表性的 AI 软件基本都会统一一个主题色（也有助于树立品牌效应）

ChatX 的 icon 和界面

# 退出及最小化反直觉

按退出键后会有悬浮窗持续存在

最小化时则不存在悬浮窗

** 按直觉来说退出之后应该软件本体包括后台进程全部退出，最小化则会最小化为悬浮窗。** 现在这样的操作逻辑容易使人不知道如何完全退出软件。

设置面板可以与个人中心分离单拎出来放在左下角，具体参考

# 文生文方面

语言模型的调试暂不谈论，旁边的 Live2D 有几点

首先是鼠标跟随，只有头部转动，眼部不转动，效果及其诡异

可以参考以下 Live2D

另：可以给灵悉设计一个看板娘，整体聊天的 Live2D 可以偏向二次元风格一点而非真人风格，比较迎合年轻游戏玩家爱好吧
